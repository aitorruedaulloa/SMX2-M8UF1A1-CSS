# CSS

##Breve historia y utilidades
La traducción de acrónimo es **hojas de estilo en cascada**, es la parte que se enfoca en el estilo al otro lado de HTML. 

Sin CSS podemos diseñar una página web claramente, pero sería plana en cuánto a estilo y muy aburrida, para eso está esté 
tipo de lenguaje que ayuda a HTML a que nuestra página web se vea más vistosa. 


Hace falta aclarar que no exactamente un
lenguaje de programación en si cómo puede ser HTML sino que és más un lenguaje para estilo de hojas. Es decir no nos permite crear,
sino diseñar el código que hemos picado con HTML
